## Describing the matrix

```javascript
def describe_mat(matrix):
    print(f'Matrix:\n{matrix}\n\nShape:\t{matrix.shape}\nRank:\t{matrix.ndim}\n')
```

This is a function that can describe the shape, rank, and rank of the matrix

### Matrix example:

```javascript
A = np.array([
    [1, 2],
    [3, 1]
])
describe_mat(A)
```

### Output:

Matrix:

[[1 2]
 [3 1]]

Shape:	(2, 2)
Rank:	2
